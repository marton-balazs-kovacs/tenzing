# Create a step panel

Create a step panel

## Usage

``` r
ui_step_panel(number, title, help_text, content)
```

## Arguments

- number:

  Step number (1, 2, or 3)

- title:

  Step title text

- help_text:

  Help text for tooltip

- content:

  Panel content (UI elements)

## Value

HTML tag for step panel
