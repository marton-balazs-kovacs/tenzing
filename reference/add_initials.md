# Add initials to the contributors_table

This function adds the initials to the contributors_table based on the
Firstname, Middle name and Surname. The function uses whitespaces and
hypens to detect the separate names. Also, the function is case
sensitive.

## Usage

``` r
add_initials(contributors_table)
```

## Arguments

- contributors_table:

  the imported contributors_table

## Value

The function returns the contributors_table with the initials in an
additional column
